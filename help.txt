Данная программа осуществляет поиск уязвимостей смарт-контракта, которые перечислены в списке в “Настройках проверок уязвимостей”.
На данный момент реализован функционал для языка Solidity.
При проверке перед строкой, в которой найдена уязвимость, добавляется в код комментарий 
 “//SolChk…” . Там, где это возможно, код модифицируется, исправляя уязвимость.

В меню “Настройка проверок уязвимостей” можно:
-Добавить, удалить, изменить описание уязвимости.
- Управлять запуском проверок на уязвимости, изменяя значение в “Проверять?” на  “yes” / ”no”.

Чтобы добавить собственную проверку на уязвимость требуется :
- Создать файл с кодом в кодировке UTF-8 в котором :
    - На входе текущий текст смарт-контракта передается в переменную text_in.
    - На выходе (после окончания проверки на уязвимость) текущий текст смарт-контракта должен быть помещен  в переменную text_out, в переменную report  записывается информация для отчета. 
        (Код проверки должен быть написан, используя язык программирования Python )
-Прописать  в “Настройки проверок уязвимостей” информацию об уязвимости и путь к файлу, содержащему код проверки, в “Файл для внешней проверки”.
